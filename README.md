# KBC Quiz Game

A console-based quiz application inspired by Kaun Banega Crorepati (KBC), developed using Java.

## Features
- Timer-based gameplay
- Multiple-choice questions
- Score tracking
- File handling for question storage
- Exception handling
- Log4j logging
- Email notifications for quiz results

## Technologies Used
- Java
- OOP
- File Handling
- Log4j
- JavaMail API
